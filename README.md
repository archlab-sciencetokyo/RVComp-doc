# RVComp-DoC

This is the repository of RVComp Documentation.

To build the documentation, please use the following commands (NOTE: this requires the use of [uv](https://github.com/astral-sh/uv) tool to manage the virtual environment):
```bash
$ uv sync
$ uv run make html
```

Documentation are available at: https://archlab-sciencetokyo.github.io/RVComp-doc/
